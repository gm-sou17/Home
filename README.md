# 🏠 Home Design Web App (Basic Project)

This is a simple web-based Home Design application created using HTML, CSS, and basic JavaScript.

⚠️ Note: This is not a complete project. It is a basic UI prototype.

---

## 🚀 Features

* 🔐 Login & Signup Page
* 🏡 Home Page UI
* 🎨 Design Selection Option
* 🏗️ Choose Home Structure (like location, parking, floors, etc.)
* 📐 Multiple Design Options

---

## 📂 Project Flow

1. User opens Login / Signup page
2. After login → goes to Home Page
3. Click on **Design Button**
4. Redirect to Design Page
5. Select different house structures and layouts

---

* JavaScript (Basic)
